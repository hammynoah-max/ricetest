# AGENTS.md — 《쌀은 아직 밥이 아니다》

## 프로젝트 목적
연작 장편소설 《쌀은 아직 밥이 아니다》의 작품 원칙서와 설계 워크북을 점진적으로 발전시킨다.

작품의 중심축은 특정 인간이 아니라, 연구실의 볍씨가 논·수확·건조·도정·유통·부엌을 지나 따뜻한 밥 한 공기가 되는 생애다. 각 편의 인간 주인공은 볍씨를 잠시 맡아 자신의 선택과 타협을 남긴 뒤 다음 사람에게 넘긴다.

## 절대 원칙
1. 계몽소설·산업 홍보물로 만들지 않는다.
2. 전문지식은 설명보다 사건, 작업, 논쟁, 실패, 선택의 대가로 드러낸다.
3. 모든 인물은 자기 입장에서 합리적이어야 한다. 편리한 악인을 만들지 않는다.
4. 각 편에는 선택, 대가, 볍씨의 상태 변화, 다음 편에 남는 결과가 있어야 한다.
5. 볍씨와 쌀을 말하거나 생각하는 인격체로 직접 의인화하지 않는다.
6. 확인되지 않은 사실을 단정하지 않는다.
7. 자료가 부족하면 `[확인 필요]`, `[작가 결정 필요]`, `[현장 취재 필요]`로 표시한다.
8. 원자료와 작가 작성본은 덮어쓰지 않는다.
9. 한 루프에서 전체 구조의 20% 이상을 동시에 바꾸지 않는다.
10. 작가 대신 최종 미학적 결정을 확정하지 않는다.

## 상태 태그
- `[확정]` 이후 임의 변경 금지
- `[가설]` 개선 제안
- `[보류]` 작가 판단 필요
- `[확인 필요]` 사실 검증 필요
- `[현장 취재 필요]` 실제 경험·인터뷰 필요
- `[삭제 후보]` 중복 또는 기능 약함
- `[충돌]` 다른 설정과 모순

## 평가 기준
각 항목을 10점 만점으로 평가한다.
- 콘셉트 선명도
- 볍씨 생애의 연속성
- 단편 간 인과성
- 인물 입체성
- 갈등과 선택의 강도
- 정보의 장면화
- 미식적 신선함
- 사실 신뢰도
- 정서적 여운
- 출판·독자 매력

## 종료 기준
- 중심 질문과 한 문장 소개가 일치한다.
- 볍씨 이동 단계가 빠짐없이 연결된다.
- 모든 핵심 편에 선택·대가·다음 편의 결과가 있다.
- `[충돌]`이 0개다.
- 작가 판단 항목이 명확한 질문 목록으로 정리된다.
- 최근 2개 루프의 총점 개선이 3점 미만이다.
- 더 자동 보완하면 작품의 개성이 약해질 위험이 있다.
