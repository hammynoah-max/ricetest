# 루프 시작 전 입력 감사

## 읽은 지시 파일
- `AGENTS.md`
- `CODEX_MASTER_LOOP_PROMPT.md`
- `RUNBOOK.md`
- `state/PROJECT_STATE.md`
- `state/SCORECARD.md`
- `state/CHANGELOG.md`

## 현재 읽을 수 있는 파일
- `AGENTS.md`
- `CODEX_MASTER_LOOP_PROMPT.md`
- `RUNBOOK.md`
- `state/PROJECT_STATE.md`
- `state/SCORECARD.md`
- `state/CHANGELOG.md`

## 현재 찾을 수 없는 필수 입력
- `workbook/` 폴더
- `workbook/` 폴더의 최신 작품 원칙서 또는 워크북
- `source/` 폴더
- `source/` 폴더의 원자료

## 추출 상태
- 첨부 zip은 `codex_loop_package/rice_novel_codex_loop/` 아래로 추출됨.
- zip 내부에는 `workbook/`, `source/`, `drafts/`, `reviews/` 폴더가 포함되어 있지 않았음.
- 결과 저장 규칙 준수를 위해 `drafts/`, `reviews/`, `state/` 폴더는 새로 생성함.

## 진행 판단
- 루프 1의 핵심 입력인 작가 작성본과 원자료가 없어, 현황 해부를 수행할 근거가 부족함.
- 루프 2, 루프 3은 루프 1 진단과 최신 워크북/원자료 확인을 전제로 하므로 현재 자동 진행하지 않음.
- 부족한 내용을 창작하지 말라는 원칙 및 즉시 정지 조건에 따라, 추가 입력이 필요함.

## 필요한 사용자 제공 자료
1. `workbook/` 폴더 또는 최신 작품 원칙서/워크북 파일
2. `source/` 폴더 또는 원자료 파일 묶음
3. 이전에 작성한 작가 확정 사항이 있다면 해당 파일
