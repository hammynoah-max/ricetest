# 밥취향연구소

기획서 `밥취향찾기_웹서비스_최종종합기획서_v1.0.md`의 P0 범위만 구현한 모바일 우선 웹서비스입니다.

## 포함 범위

- 랜딩과 진단 안내
- 9문항 취향 테스트
- localStorage 기반 응답 저장과 복원
- 점수 계산, 결과 유형 판정, 품종 추천
- 5개 결과 유형과 결과 카피
- 12개 품종 데이터 기반 2~3개 탐색 후보
- 다시하기
- 모바일 360px 완주

## 제외 범위

로그인, DB, 관리자 화면, 쇼핑 기능, AI 채팅은 넣지 않았습니다. 기획서상 P1/P2인 결과 링크 공유, 취향 지도, 품종 상세 바텀시트, about 페이지, 분석 이벤트도 이번 P0 구현에서는 제외했습니다.

## 실행

```bash
npm install
npm run dev
```

## 테스트

```bash
npm test
```

## 데이터 수정

- 질문 문구: `src/data/questions.ts`
- 결과 카피: `src/data/resultTypes.ts`
- 품종 좌표와 별점: `src/data/riceVarieties.ts`
- 공통 문구와 서비스명: `src/data/copy.ts`

품종 좌표와 별점은 최종 실측 산출물이 아직 없으므로 잠정값입니다. 앱 화면 코드를 고치지 않고 데이터 파일에서 교체할 수 있습니다.
